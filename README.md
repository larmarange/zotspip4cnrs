zotspip4cnrs
============

ZotSpip &amp; co pour le kit CNRS.

Ce plugin [SPIP](http://www.spip.net/fr) permet de faire fonctionner les plugins [ZotSPIP](http://contrib.spip.net/ZotSpip), [BiblioCheck](http://contrib.spip.net/Biblio-Check-1-0) et [Bibliographie AERES](http://contrib.spip.net/Bibliographie-AERES-2-0) avec le [kit CNRS](http://www.harmoweb.cnrs.fr/).

Pour l'installer, vous pouvez utiliser la fonction *Télécharger un plugin depuis son archive* de SPIP en spécifiant l'URL suivante : https://github.com/larmarange/zotspip4cnrs/archive/master.zip

**ATTENTION :** ce plugin en cours de développement. Le plugin est pleinement fonctionnel mais le rendu esthétique n'est pas optimal.
